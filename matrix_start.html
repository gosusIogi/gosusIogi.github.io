matrix server

Сервер для общения matrix.raspad.space

Что это такое?

Matrix — это открытый протокол для децентрализованного общения в реальном времени. Он может быть использован для текстовых сообщений, групповых чатов, аудио/видео звонков и создания ботов. Разрабатывается энтузиастами из Matrix.org Foundation с 2014 года.

Будучи открытым стандартом, Matrix объединил огромное количество энтузиастов. Это означает, что всегда будет хорошая группа людей, увлечённых поддержанием Matrix в рабочем состоянии.

Спецификации протокола, равно как и клиент-серверная часть находятся в открытом доступе, в отличие от Telegram, который в настоящее время в открытом доступе держит только свой клиент, что вызывает немало вопросов по поводу безопасности использования протокола MTProto.

Quick start guide

Шаг первый

Качаем один из многочисленных клиентов, представленных тут.

Наиболее предпочтительный клиент Element.io. Браузерная версия, не требующая установки доступна на app.element.io.

Второй по предпочтительности клиент fluffychat.

Шаг второй

Открываем Element и нажимаем начать. В окне выбора сервера выбираем пункт "другой".



Шаг третий

Вводим адрес сервера matrix.raspad.space и нажимаем "продолжить".



Шаг четвертый

Нажимаем "зарегистрироваться". Придумываем логин и пароль, жмем "далее" и входим в систему.


Quick start desktop

Шаг первый

Заходим в веб версию app.element.io и нажимаем "Создать учетную запись".



Шаг второй

Нажимаем кнопку "Изменить" в верхней части формы регистрации.



Шаг третий

Выбираем "другой домашний сервер", вводим matrix.raspad.space и нажимаем "Продолжить".



Шаг четвертый

Придумываем логин и пароль, жмем "Зарегистироваться" и входим в систему.
